◆ CollisionGroupIndex

short Engine.Body.CollisionGroupIndex = 0

Optionally, if you don't want to use the CollisionCategorybits and CollisionMaskBits, you can use the group index.

Objects with the SAME POSITIVE group index will collide. Objects with the SAME NEGATIVE group index won't collide. Objects with both a NON-ZERO group index, but DIFFERENT indices, will use the category/mask bits Objects with ZERO group index will use the category/mask bits
