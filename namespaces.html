 CCacheGeneric cache system which allows caching of a user-specified type inheriting from CacheElement. The cache will manage an internal hash list of elements associated with their key, and provides a LookUp and Insert function to interface with the cache. Cache elements have a ref count to count if they're still in use, and will automatically get removed from the cache if not. In this case, CacheElement.DestroyElement is called, which can be overwritten by the user.
