Detailed Description

Handles the audio management for the game, including mixing. Expects all audio files to be in "ProjectFolder/Content/Sounds".

Difference between Music and Sound effects: Music should be an.mp3 file, only has a volume A sound effect should be a .wav file and has a channel to play in.
