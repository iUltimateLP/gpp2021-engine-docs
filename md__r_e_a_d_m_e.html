Structure

The project follows the following structure:

  • WiiNotFit.sln
  • WiiNotFit/
    • Engine/
      • This contains the engine CS project, including all SDL functionality
    • Test/
      • Contains a very simple test game project utilizing the engine
    • ...

Any other game using the engine is placed in the WiiNotFit.sln solution and references the Engine project:

Code Style

The project inherits the official Microsoft C# Code Style. Despite that, we're using Tabs instead of Spaces, so make sure to set Visual Studio to use tabs here:

Extras -> Optionen -> Text-Editor -> Alle Sprachen -> Tabstopps and set the tab sizes to 4 and to retain tab stops.

SDL2.dll dependency

The project requires SDL2 and it's runtime binaries. It is supplied in the root folder of this repository, or can be downloaded here:

  • SDL (main SDL library)
  • SDL_image (SDL image library)
  • SDL_mixer (SDL audio mixer library)
  • SDL_ttf (SDL TrueType font library)
